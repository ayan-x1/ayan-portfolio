````markdown
## 🧭 Installation

Clone & run locally:

```bash
git clone https://github.com/ayan-x1/ayan-portfolio.git
cd ayan-portfolio
npm install
````

---

## 🚀 Usage

Start dev server:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the app.

Build & start production:

```bash
npm run build
npm run start
```

